HOMEWORK - SPRING 2025

HOMEWORK 2 -  GRADING KEY

1. (14 points) Documentation and coding style:

  • Name, ID number and recitation at the top of ALL java files [2 points]
  • Precise and correct specifications[4 points]
  • Indentation[2 points]
  • Variable names[2 points]
  • Javadoc Style *(see NOTE). [4 points]

NOTE 1: You only need to submit your *.java files. We will use the javadoc *.java command to create the javadoc files.

NOTE 2: YOUR ASSIGNMENT WILL NOT BE GRADED if you submit only *.class files.

NOTE 3: YOUR ASSIGNMENT WILL NOT BE GRADED if you use ArrayList, Vector, Java API LinkedList or any predefined Java API Data Structures.

2. (18 points) Source code:
  • Implementation done as specified (additional methods are ok)[10 points]
  • Throw an exception back to the calling method(s) if a precondition is violated.[8 points]
3. (61 points) Program correctness:
  • Compiles without error(no partial credit) [4 points]
  • The methods insertInfo, removeAllPurchased, moveItem, printAll, printByLocation, cleanStore,and checkOut work correctly:
    • Inserts item info correctly into list.[8 points]
    • Does not insert item info if data is invalid.[2 points]
    • Removes all checked out items from the list.[8 points]
    • Correctly moves items from one location to another.[6 points]
    • Ignores move command if the data location or RFID is invalid or item isn't found.[2 points]
    • Prints inventory list in RFID(alphabetical) order.[8 points]
    • Prints items from one location in RFID(alphabetical) order.[4 points]
    • Clean store command works correctly.[6 points]
    • Checkout command works correctly [6 points]
    • Checking for incorrect values(RFID, price, starting location).[7 points]
4. (7 points) Correct analysis of Complexity of the code.
  • Using big-O to include the order of complexity for the methods insertInfo, removeAllPurchased, moveItem, printAll, printByLocation, cleanStore,and checkOut[1 point each].

5. (2 points) Extra credit:

  • Prints all items that have the given rfidTagNumber.[2 points]
