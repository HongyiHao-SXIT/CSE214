HOMEWORK 5 - GRADING KEY

1. (17 points) Documentation and coding style:

  • Name, ID number and recitation at the top of ALL java files [2 points]
  • Precise and correct specifications[4 points]
  • Indentation[2 points]
  • Variable names[2 points]
  • Javadoc Style *(see NOTE). [4 points]

NOTE 1: You only need to submit your *.java files. We will use the javadoc *.java command to create the javadoc files.

NOTE 2: YOUR ASSIGNMENT WILL NOT BE GRADED if you submit only *.class files.

2. (12 points Points) Source Code
  • Implementation done as specified (additional methods are okay). [10 points]
  • Correct error handling and error checking as specified. [4 points]
3. (70 Points) Program Correctness
  • Program compiles without any errors. [10 points]
  • Program searches for a previously saved mailbox.obj and loads it [2 points]
  • Program properly saves mailbox.obj using Serializable on exit. [2 points]
  • Program properly handles malformed user input. [2 points]
  • Main menu options work correctly: [30 points]
    • Add folder – prompt for folder name, create and add folder [4 points]
    • Remove folder – prompt for folder, remove that folder if it exists [4 points]
    • Compose email – prompt for TO, CC, BCC, Subject, Body. Create new email, add to inbox [7 points]
    • View folder – prompt for folder name, opens that folder’s submenu if it exists [4 points]
    • View inbox – opens the folder submenu for the inbox [2 points]
    • View trash – opens the folder submenu for the trash [2 points]
    • Empty trash – empties the trash folder of all its emails [4 points]
  • Folder submenu options work correctly: [22 points]
    • Move email – prompt for email index, displays folder list, prompts for destination folder. Can move to Inbox, trash, current folder [7 points]
    • Delete email – prompt for email index, move to trash [4 points]
    • View email contents – prompt for email index, prints that email info [4 points]
    • Sort by subject ascending/descending – sorts correctly [4 points]
    • Sort by date ascending/descending – sorts correctly [4 points]
4. (4 points) Extra Credit
  • Use Graphical User Interface (GUI) for program input and display the results as mentioned in the homework description. [4 points]
