HOMEWORK 4 - SPRING 2025

HOMEWORK 5 - GRADING KEY

1. (14 points) Documentation and coding style:

  • Name, ID number and recitation at the top of ALL java files [2 points]
  • Precise and correct specifications[4 points]
  • Indentation[2 points]
  • Variable names[2 points]
  • Javadoc Style *(see NOTE). [4 points]

NOTE 1: You only need to submit your *.java files. We will use the javadoc *.java command to create the javadoc files.

NOTE 2: YOUR ASSIGNMENT WILL NOT BE GRADED if you submit only *.class files.

NOTE 3: YOUR ASSIGNMENT WILL NOT BE GRADED if you use ArrayList, Vector or any predefined Java API Data Structures.

2. (8 Points) Source Code
  • Implementation done as specified (additional methods are okay). [8 points]
3. (78 points) Program Correctness
  • Program compiles without any errors. [14 points]
  • Creation of tree is correct. [13 points]
  • Traversal of tree is correct. [11 points]
  • Main Menu is displayed at the appropriate times (L, H, T, Q options). [8 points]
  • Questions and Answer menu is correct. [14 points]
  • Choice of answer leads to the correct subtree of questions. [14 points]
  • Error Checking and special cases: [2 points each = 4 points]
    • Blank lines in the file ignored.
    • Error messages are clear and understandable.
4. (8 Points) Extra Credit
  • Program allows for a 9-ary tree to be used. [2 points]
  • Program allows the user to go backwards in the sequence of questions. [2 points]
  • Creative idea that your program does (documented in the driver class!). [up to 4 points, decided by your TA]
