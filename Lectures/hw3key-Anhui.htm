HOMEWORK - SPRING 2025

HOMEWORK 3 -  GRADING KEY

1. (14 points) Documentation and coding style:

  • Name, ID number and recitation at the top of ALL java files [2 points]
  • Precise and correct specifications[4 points]
  • Indentation[2 points]
  • Variable names[2 points]
  • Javadoc Style *(see NOTE). [4 points]

NOTE 1: You only need to submit your *.java files. We will use the javadoc *.java command to create the javadoc files.

NOTE 2: YOUR ASSIGNMENT WILL NOT BE GRADED if you submit only *.class files.

2. (32 points) Source code:
  • Implementation done as specified (additional methods are okay). [8 points]
  • Program compiles without any errors. [14 points]
  • RequestQueue class is correctly defined as a subclass of ArrayList (or other selected class). [10 points]
3. (54 points) Program correctness:
  • Output generated by program is truly random. [10 points]
  • Averages are rounded to two decimal places. [5 points]
  • Error checking is done on all inputs to the simulation. [5 points]
  • Results produced by program are reasonable:
    • A larger number of floors will generally result in larger wait times. [17 points]
    • Results are generally within the expected bounds of the test cases. [17 points]

4. (4 points) Extra credit:

  • Elevators can carry an unlimited number of passengers and can move in any direction(up or down).[4 points]
