HOMEWORK - SPRING 2025

Homework 1 -  GRADING KEY

1. (17 points) Documentation and coding style:

  • Name, ID number and recitation at the top of ALL java files [2 points]
  • Precise and correct specifications[4 points]
  • Indentation[2 points]
  • Variable names[2 points]
  • Javadoc Style *(see NOTE). [4 points]

NOTE 1: You only need to submit your *.java files. We will use the javadoc *.java command to create the javadoc files.

NOTE 2: YOUR ASSIGNMENT WILL NOT BE GRADED if you submit only *.class files.

NOTE 3: YOUR ASSIGNMENT WILL NOT BE GRADED if you use ArrayList, Vector or any predefined Java API Data Structures.

2. (18 points) Source code:
  • Implementation done as specified (additional methods are OK)[10 points]
  • Throw an exception back to the calling method(s) if a precondition is violated.[8 points]
3. (60 points) Program correctness:
  • Compiles without error(no partial credit) [12 points]
  • The methods clone, equals, addSong, getSong, removeSong, printAllSongs, size, and getSongsByArtist, work correctly.[6 points each]
4. (4 points) Program efficiency: size and getSong should be O(1) and all other methods should be O(n)

5. Extra Credit:

  • Provide an option to play the song. For this option, you should input the song's filename along with other information related to each song (title, artist, duration, etc.). [4 points]
  • Add the menu options N, V, D, C, E and D to create and manage multiple playlists: [4 points]
